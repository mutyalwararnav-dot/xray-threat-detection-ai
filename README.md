# X-Ray Threat Detection AI 🚀

## 📌 Overview
This project is an AI-based X-ray threat detection system using Vision Transformer (ViT). It detects potential threats in X-ray images and returns predictions with confidence scores.

## 🛠 Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
- Firebase

## ⚙️ Features
- ViT-based image classification
- Training pipeline (`train.py`)
- Inference system (`test_inference.py`)
- Firebase API deployment

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Train model:
python train.py

3. Run inference:
python test_inference.py

## 📁 Project Structure
- train.py
- test_inference.py
- inspect_data.py
- firebase/
- .gitignore
